NCF 2023 Stage-based competency outcomes Module A Competency taxonomy: Domain → Strand → Sub-skill → Observable Indicator
NCF 2023 Experiential and play-based learning pedagogy Module D All intervention activities are play-based, experiential, low-cost
NCF 2023 Constructivist learning model Module D Peer learning groups, manipulative activities, inquiry-based tasks
NCF 2023 Formative, ongoing assessment Module B, C Monthly diagnostic cycle; continuous skill tracking; growth curves
PARAKH Diagnostic, not judgmental assessment Module B, E All assessments are diagnostic. Results are shared only for learning improvement.
PARAKH Balanced assessment blueprint Module B AI generates balanced difficulty mix: Easy / Medium / Hard
PARAKH No ranking culture in schools Module E, F Progress cards show only skill level (G/A/R). No percentages or class ranks.
PARAKH Competency mapping and tagging Module A, B Every assessment item tagged to NCF competency code (L1.1, N2.1, etc.)
