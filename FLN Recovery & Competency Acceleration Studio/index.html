Assign students to intervention groups based on competency levels
🔴 Group A – Foundational Alert
Letter & Sound Recognition Focus · Level: Below Foundational

Students scoring below 3/10 in letter recognition or number recognition

  • Alphabet chart tracing and oral repetition
  • Phoneme-grapheme mapping with manipulatives
  • One-to-one number correspondence activities
  • Picture-word matching games
  • Daily 5-minute oral reading with teacher
  • Number line walking activity (1–20)
  • Storytelling with picture cards
🟠 Group B – Emerging Learner
Word Reading & Basic Comprehension · Level: Emerging

Students scoring 3–6/10 in literacy or 3–6/10 in numeracy

  • Guided reading with decodable texts
  • Word wall activities and sight-word practice
  • Addition and subtraction with concrete materials
  • Comprehension question-answer exercises
  • Place value with bundles of sticks (tens & ones)
  • Short paragraph reading aloud daily
  • Number pattern completion (skip counting)
🟢 Group C – Grade Approaching
Comprehension & Operations Reinforcement · Level: Near Grade-Ready

Students scoring 6–8/10 requiring consolidation and acceleration

  • Paragraph comprehension with inference questions
  • Multi-step word problem solving strategies
  • Independent reading with comprehension journals
  • Mental math and estimation challenges
  • Peer tutoring and collaborative learning tasks
  • Creative writing with grade-level vocabulary
  • Data interpretation and simple graphs

🏛️ NEP 2020 Framework
NEP 2020 & NCF 2023 Reference
Policy framework, FLN goals, and reform milestones
National Education Policy 2020 – FLN Mission
NEP 2020 · Section 2.1–2.7

NEP 2020 places Foundational Literacy and Numeracy as the most critical and urgent priority of the entire education system. The policy recognises that if children do not attain FLN by Grade 3, the learning gap widens and becomes extremely difficult to close by later grades.

NIPUN Bharat Mission: The National Initiative for Proficiency in Reading with Understanding and Numeracy (NIPUN Bharat) was launched in 2021 with the target: every child by Grade 3 (by 2026–27) achieves basic reading and arithmetic competency.

  • Reading with comprehension by end of Grade 3
  • Basic arithmetic operations by end of Grade 3
  • Competency-based progression (not age-based social promotion)
  • Early intervention for all children showing foundational gaps
  • Teacher professional development in FLN pedagogy
  • Integration of play-based and activity-based learning
  • Multilingual approaches to literacy instruction
NCF 2023 – Foundational Stage (Ages 3–8)
NCF 2023 · Foundational Stage

The National Curriculum Framework for the Foundational Stage (NCF-FS) 2022 provides specific competency descriptors for each domain. It emphasises a play-based, inquiry-driven, child-centred pedagogy that builds foundational skills without rote learning or excessive drilling.

  • Literacy: Oral language → Print awareness → Phonological awareness → Decoding → Comprehension
  • Numeracy: Number sense → Operations → Spatial reasoning → Pattern recognition
  • Cognitive, social-emotional, physical, and cultural development integration
  • Mother-tongue first approach with gradual multilingual transition
  • Formative assessment using observation, portfolios, and conversations
  • Inclusion of all children including those with special learning needs
Competency Benchmarks by Grade
NIPUN Bharat · FLN Benchmarks

Grade 1 Literacy: Identify letters, basic phonemes, read simple CVC words, write own name

Grade 2 Literacy: Read simple sentences, answer literal comprehension questions, write short sentences

Grade 3 Literacy: Read short paragraphs fluently (45–60 wpm), infer meaning, write short paragraphs

Grade 1 Numeracy: Count to 99, compare numbers, add/subtract within 20

Grade 2 Numeracy: Read/write numbers to 999, understand place value, add/subtract 2-digit numbers

Grade 3 Numeracy: Multiply single-digit numbers, understand fractions, solve simple word problems

FLN Analytics Formulas Used in This App
Deterministic Local Formulas

Foundational Gap Score (FGS):
FGS = ((Lit_Score × 0.5) + (Num_Score × 0.5)) × 10
Where Lit_Score and Num_Score are averages of sub-domain ratings (0–10)

Gap Severity Index (GSI):
GSI = (1 - (FGS/100)) × 100 × Attendance_Weight
Attendance_Weight = 1 + (1 - Attendance/100) × 0.3

Acceleration Velocity (AV):
AV = (Current_Score - Baseline_Score) / Weeks_Elapsed

Catch-Up Probability (CUP):
CUP = MIN(100, (AV × Remaining_Weeks / Gap_to_Close) × 100)

Early Warning Index (EWI):
EWI = (Lit_Gap% × 0.3) + (Num_Gap% × 0.3) + (1-Attendance%) × 20 + Gender_Gap% × 0.5 + Loss_Months × 1.5

Recovery Success Projection (RSP):
RSP = MIN(95, FGS + (AV × Plan_Weeks × 1.2))

APK Conversion Guide
PWA → Android APK

Step 1 – Build Static Files: Save index.html, manifest.json, service-worker.js, and author.jpg in one folder (e.g. /fln-studio-app)

Step 2 – Test Locally: Use VS Code Live Server or Python: python -m http.server 8080 — open http://localhost:8080

Step 3 – Host Free Online: Upload to GitHub Pages (github.com → New Repo → Upload files → Settings → Pages) or Netlify Drop (drop.netlify.com)

Step 4 – Convert to APK: Go to PWABuilder.com → Enter your hosted URL → Download Android package → Open in Android Studio → Build APK

Step 5 – Alternative (WebView APK): Create new Android Studio project → Add WebView → Set URL to your GitHub Pages link → Add internet permission → Build APK → Install on device

Step 6 – Offline APK: Bundle all files in /assets/www/ folder in Android project → Set WebView to load from file:///android_asset/www/index.html → No internet required
