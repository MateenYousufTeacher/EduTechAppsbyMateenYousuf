## 🚀 How to Run Locally

### Option 1: Open directly
1. Download all 3 files into one folder
2. Double-click `index.html` → Opens in browser

### Option 2: VS Code Live Server
1. Open folder in VS Code
2. Install "Live Server" extension
3. Right-click `index.html` → "Open with Live Server"

### Option 3: Python local server
```bash
cd fln-diagnostic-portal
python3 -m http.server 8080
# Open: http://localhost:8080
```

## 🌐 Deploy FREE Online

### GitHub Pages (Recommended)
1. Create a GitHub account (free)
2. Create new repository named `fln-portal`
3. Upload all 3 files
4. Go to Settings → Pages → Source: main branch
5. Your URL: `https://yourusername.github.io/fln-portal`

### Netlify
1. Go to netlify.com
2. Drag-and-drop the folder
3. Get instant free URL

### Cloudflare Pages
1. Go to pages.cloudflare.com
2. Connect GitHub repo
3. Deploy automatically

---

## 📱 Install as Mobile App (PWA)
1. Open the website on phone browser
2. Tap "Add to Home Screen" (appears automatically)
3. App installs like a native app — works offline!

## ✅ Features
- Grade 1, 2, 3 diagnostics for Language & Numeracy
- Oral + Written question mix
- NEP 2020 & NCF 2023 aligned competencies
- Visual skill-gap analyzer (bar charts, RAG indicators)
- Remedial suggestions (classroom, peer, home, play-based)
- Holistic Progress Card generator
- PDF print/download for all outputs
- Works offline (PWA)
- Mobile-first responsive design
- 100% FREE — no login, no backend, no database
